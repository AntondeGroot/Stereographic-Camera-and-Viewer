# Stereographic Camera and Viewer

Take and view 3D images and send them automatically to Lightroom or some different folder.

## Bill of Materials Camera Rig
- Aluminium square profile 10x10x1mm (or thicker if you like) 
- Aluminum Square profile 15x15x1.5mm (if you want to be able to extend it) the 10x10mm profile should be able to fit inside it.
- Raspberry Pi Zero 2W
- LCD Screen: Mini HDMI-compatibel display 3,5 inch for raspberry pi e.g. aliexpress.com/item/1005003573751563.html
- Battery: Pisugar 3 5000mah or powerbank (5v?)
- 2x data cables (e.g. micro-b usb to usb-a) (check your own camera what is required)

#### trigger cable

To trigger a photo you only need to connect the wire that goes to the tip of the jack, to the wire that connects to the sleeve of the jack.

- 2x trigger cables, e.g. for Nikon D5300: N3 trigger cables to 2.5mm 3 pole jack female
- 1x 2.5mm Jack splitter 2xFemale to 1xMale 3 pole cable!
- Reset button (Normally Open)
- Heatshrink

2 DSLR's with the same lens. Preferably internal zoom lenses.
